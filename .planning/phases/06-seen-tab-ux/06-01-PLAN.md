Create the file `scripts/seen-leads.gs` with a complete Google Apps Script containing two functions:

**1. `onEdit(e)` — simple trigger (per D-03)**

Logic:
- Guard: exit early if `e` is undefined or `e.range` is undefined (programmatic edits sometimes lack event object)
- Get the edited sheet name via `e.range.getSheet().getName()`
- Guard: exit early if sheet name is "Seen" (per D-05, never process edits on the Seen tab itself)
- Guard: exit early if sheet name is "Sheet1" (ignore default tab — Claude's discretion)
- Guard: exit early if edited column is not 6 (column F — per D-01)
- Guard: exit early if edited row is 1 (skip header row)
- Guard: exit early if `e.value` is not "TRUE" (only act when checkbox is checked — per D-04)
- Get the active spreadsheet via `SpreadsheetApp.getActiveSpreadsheet()`
- Get or create the "Seen" tab:
  - Try `ss.getSheetByName("Seen")`
  - If null, create it: `ss.insertSheet("Seen")` and write header row `["Post ID", "Author", "Post Link", "Post Date", "Scraped At", "Source Group"]` to row 1 (per D-08, D-09)
- Read columns A-E from the edited row: `sourceSheet.getRange(row, 1, 1, 5).getValues()[0]`
- Build the Seen row: append the source sheet name as column 6 (per D-06, D-07): `[...rowData, sourceSheetName]`
- Append to Seen tab: `seenSheet.appendRow(seenRow)`
- Delete the original row from the source sheet: `sourceSheet.deleteRow(row)` (per D-04)

**2. `addCheckboxesToAllGroups()` — one-time utility (per D-02)**

Logic:
- Get all sheets in the spreadsheet
- For each sheet, skip if name is "Seen" or "Sheet1"
- Get the last row via `sheet.getLastRow()`
- If lastRow < 2, skip (no data rows)
- Insert data validation (checkbox) on range F2:F{lastRow}:
  ```
  var rule = SpreadsheetApp.newDataValidation().requireCheckbox().build();
  sheet.getRange(2, 6, lastRow - 1, 1).setDataValidation(rule);
  ```
- Also set the F1 header to "Seen" on each group tab:
  ```
  sheet.getRange(1, 6).setValue("Seen");
  ```

Add a JSDoc comment at the top of the file:
```
/**
 * Seen Leads UX — Google Apps Script

 * Attached to the Facebook Leads Google Sheet.
 *
 * onEdit(e): Moves a lead row to the "Seen" tab when its column F checkbox is checked.
 * addCheckboxesToAllGroups(): One-time utility to add checkboxes to all existing lead rows.
 *
 * Deployment: Paste into Extensions > Apps Script on the target sheet.
 * After pasting, run addCheckboxesToAllGroups() once from the script editor.
 */
```
