- Name it: "Apify Scrape"
- Method: POST
- URL: `https://api.apify.com/v2/acts/apify~facebook-groups-scraper/run-sync-get-dataset-items`
- Authentication: Predefined Credential Type -> Header Auth
  - Create or select credential named "Apify API"
  - Header Name: `Authorization`
  - Header Value: `Bearer <your-apify-token>` (stored in n8n credential store)
- Body Content Type: JSON
- Specify Body: Using JSON
- JSON Body (use expressions):
```json

  "resultsLimit": {{ $json.post_count }},
  "viewOption": "CHRONOLOGICAL"
}
```
- Options > Timeout: 300000 (5 minutes -- sync endpoint can take 60-300s)
- Options > Response Format: JSON (auto-parse)

**Step 3 -- Connect:**
- Wire: Execute Workflow Trigger -> Apify Scrape

In the leads_scrape_apify workflow in n8n:

**Step 1 -- Code node (field mapping):**
- Add node: "Code"
- Name it: "Map Fields"
- Language: JavaScript
- Mode: Run Once for All Items
- Code:
```javascript
const items = $input.all();

if (items.length === 0) {
  return [];
}

return items.map(item => ({
  json: {
    postId: item.json.postId ?? item.json.id ?? '',
    authorName: item.json.authorName ?? item.json.author ?? '',
    postText: item.json.message ?? item.json.text ?? '',
    postLink: item.json.url ?? item.json.postUrl ?? '',
    postDate: item.json.time ?? item.json.timestamp ?? '',
  }
}));
```
- Wire: Apify Scrape -> Map Fields

**Step 2 -- IF node (zero-post check):**
- Add node: "IF"
- Name it: "Has Posts?"
- Condition: Number -> `{{ $input.all().length }}` is greater than 0
  - Alternatively use Expression mode: `{{ $input.all().length > 0 }}`

**Step 3 -- True branch (posts exist):**
- The true output goes directly to the workflow output (no additional node needed -- n8n returns all items reaching the end)

**Step 4 -- False branch (zero posts):**
- Add node: "Code" on the false branch
- Name it: "Log Empty"
- Code:
```javascript
const groupUrl = $('Execute Workflow Trigger').item.json.group_url;
console.log(`No posts found for ${groupUrl}`);
return [{ json: { posts: [] } }];
```
- This ensures an empty array is returned to the calling workflow

**Step 5 -- Connect the full chain:**
- Execute Workflow Trigger -> Apify Scrape -> Map Fields -> Has Posts?
- Has Posts? (true) -> [end of workflow, items pass through]
- Has Posts? (false) -> Log Empty -> [end of workflow]

**Step 6 -- Save workflow.**

Per SCRP-03: Code node maps Apify fields to { postId, authorName, postText, postLink, postDate } with fallback aliases.
Per SCRP-04: IF node detects zero posts, logs message, passes empty array downstream.
